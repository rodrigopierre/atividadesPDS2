PDS2 - Projeto Final Documentation

Bem-vindo à Documentação do Trabalho prático de PDS2!

Esta documentação descreve o sistema de jogos de tabuleiro, com cadastro e amarzenamento de estatísticas de usuários.

  • Cadastro de jogadores e armazenamento de dados: As classes Register e Player são responsáveis por essa função.
  • Lógica de jogos: A clase BaseGame, junto com as filhas Reversi, Lig4 e JogoDaVelha possuem os métodos necessários para jogar.
  • Execução do Sistema: O ExecutionSystem e a Main são responsáveis por organizar o menu e o fluxo de métodos para cada função do Menu.

Como executar o programa:

Para rodar o programa, siga os seguintes passos na raiz do projeto

    • Compilar o código:

      • Execute o comando:

        make

        no terminal para compilar todos os arquivos do projeto.

    • Rodar o Programa:

      • Após a compilação, execute o comando:

        bin/main

        para iniciar os jogos!
